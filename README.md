# 点呼をする

## 動作について
　**実行すると data と scans の2つのフォルダが生成されます．**  
　テキストボックスに入力するか，それ以外でも常にキー入力を受け付けています．  
　入力はエンターキーごとに処理します．  
　10桁または5桁以外の数字が入力された場合はスキャン失敗と表示されます．  
　scans フォルダ内にバイナリファイルが生成されますが，アプリ内のスキャン履歴は data 内に保存される LocalStorageです．  
　
## 問題
　履歴を削除した際，一度ウィンドウを読み直さないと input を受け付けなくなってしまっています．  
　ファイルが自動で scans 内に保存されますが，LocalStorageはこれと同期しない．削除機能は実質機能していない
　-->起動時に scans フォルダ内にデータがあったら警告するようにしたい  
 
## いくつか
　Electronを使用しているため起動は遅いです．  

　思いついたアイデアをほぼバイブコーディングで作成した（ごめんなさい）ものなので，  
　実用ではなく，こんなデザインのものを考えていたという程度に思っていただけると幸いです
