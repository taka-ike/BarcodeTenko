# 点呼をする

## 動作について
　テキストボックスに入力するか，それ以外でも常にキー入力を受け付けています．
　10桁または5桁以外の数字が入力された場合はスキャン失敗と表示されます．
　
　履歴を削除すると一度ウィンドウを読み直すか表示しなおさないと input を受け付けなくなってしまっています．

　常にファイルに書き込みたかったのですが，現在は localStorage に保存してしまっていて，ソフトを立ち上げなおしても残ってしまいます．ここら辺は構造体への私の知識不足もあって不安定です．
