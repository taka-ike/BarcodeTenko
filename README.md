# 点呼をする

## 動作について
　テキストボックスに入力するか，それ以外でも常にキー入力を受け付けています．  
　入力はエンターキーごとに処理します．  
　10桁または5桁以外の数字が入力された場合はスキャン失敗と表示されます．  
　連続して同じ数字が入力された場合に弾く機能はありません．  
　
## 問題
　履歴を削除した際，一度ウィンドウを読み直さないと input を受け付けなくなってしまっています．  

　常にファイルに書き込むのが理想ですが，現在は localStorage へ保存するようになっています．  
　AppData/Roaming/tenkoに書き込まれてしまい，アプリを立ち上げなおしても残ります．   

## いくつか
　もともと React でウェブアプリとして動いていたもので，起動は遅いです．  

　思いついたアイデアをほぼバイブコーディングでつくっただけなので，  
　実用ではなく，こんなデザインのものを考えていたという程度に思っていただけると幸いです．
